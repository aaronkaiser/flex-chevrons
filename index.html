Welcome to Flex Chevrons.

This is a small project I'm working on to provide sample HTML + CSS code for users wanting to create a wizard-type chevron with arrows that is flexible. Instead of using pixels to define the height and spacing, it uses ems.

I'll be building out this page eventually as a tutorial. Until then, please feel free to view or download my files to see what I came up with.
